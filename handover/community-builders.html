Events & Programming (Amy)

  • 10+ events per month
  • Event types: workshops, meetups, builder-led sessions, partner events
  • Builder onboarding: welcoming new community members, orientation
  • Event performance tracking: attendance, satisfaction, frequency patterns
  • Builder-led event support: logistics, scheduling, venue coordination for builder-organized sessions
  • Data collected: attendance, feedback, engagement metrics
Key principle: Amy owns execution and logistics. Laura provides strategic direction on programming content and themes. This boundary is important — Amy should be protected from carrying strategic load.
